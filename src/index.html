Авто-подтверждение действий агента

            
Когда ВКЛ — агент удаляет файлы и запускает команды без диалога подтверждения. Делает работу быстрее, но опаснее.

          

          
        

        

          

            
Показывать скрытые файлы

            
Файлы и папки начинающиеся с точки (.git, .env и т.п.)

          

          
        

        

          

            
Отключить системный прокси

            
Все API-запросы пойдут напрямую, игнорируя настройки прокси Windows / переменные HTTP_PROXY. Помогает если корпоративный прокси / VPN ломает SSL или режет соединения к OpenAI/Anthropic. Требует перезапуск приложения.

          

          
        

        

          

            
Свой прокси (опционально)

            
Если оставить пустым и «Отключить прокси» выключено — используется системный прокси. Формат: http://host:port или socks5://host:port. Игнорируется если «Отключить системный прокси» включено. Требует перезапуск.

          

          
        

      

      

        
Ключи хранятся локально, шифруются ОС (DPAPI на Windows). «Проверить» сначала пробует /models, при ошибке — минимальный chat-запрос.

        

      

      

        

          
          

            
Не подключено

            
Подключи аккаунт GitHub чтобы push/pull/commit работали без ручного ввода пароля

          

          
        

        

          

            
            
          

          

            
            
          

          

            
            
            
          

          

            Как создать токен:

            1. Нажми «Создать токен на GitHub ↗»

            2. На странице GitHub поставь нужные галочки: минимум — repo (для приватных репозиториев) и workflow (для GitHub Actions)

            3. Внизу нажми «Generate token», скопируй (он начинается с ghp_)

            4. Вставь сюда + введи свой username и жми «Подключить»

            Безопасность: токен сохраняется локально в шифрованном виде. Никуда не отправляется.
