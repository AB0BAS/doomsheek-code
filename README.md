  🧠 Агентский режим (10 инструментов)

  Модель сама вызывает инструменты через теги <tool name="…">:

  ┌────────────┬─────────────────────────────────────────────────────────────────┐
  │ Инструмент │                           Что делает                            │
  ├────────────┼─────────────────────────────────────────────────────────────────┤
  │ read_file  │ читает файл                                                     │
  ├────────────┼─────────────────────────────────────────────────────────────────┤
  │ write_file │ создаёт/перезаписывает файл                                     │
  ├────────────┼─────────────────────────────────────────────────────────────────┤
  │ edit_file  │ точечное редактирование                                         │
  ├────────────┼─────────────────────────────────────────────────────────────────┤
  │ list_dir   │ дерево директории                                               │
  ├────────────┼─────────────────────────────────────────────────────────────────┤
  │ mkdir      │ создаёт папку                                                   │
  ├────────────┼─────────────────────────────────────────────────────────────────┤
  │ delete     │ удаляет файл/папку                                              │
  ├────────────┼─────────────────────────────────────────────────────────────────┤
  │ grep       │ поиск по содержимому                                            │
  ├────────────┼─────────────────────────────────────────────────────────────────┤
  │ run        │ выполняет shell-команду со стримингом вывода в реальном времени │
  ├────────────┼─────────────────────────────────────────────────────────────────┤
  │ remember   │ пишет заметку в долговременную память                           │
  └────────────┴─────────────────────────────────────────────────────────────────┘

  Память агента (memory.json) сохраняется между сессиями — агент сам решает, что положить туда.

  Тест: «Открой папку D:\elProject, найди все вызовы fetch в .js файлах, и сохрани в памяти где какие провайдеры
  используются». Должна пройти цепочка list_dir → grep → remember.
